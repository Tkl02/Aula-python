# Aula Python

Repositório de estudos em Python com foco em evolução de aprendizado.

Este repositório foi organizado para mostrar os conteúdos praticados ao longo de cursos e atividades acadêmicas. Os projetos principais ficam em outro repositório, enquanto aqui está o registro das trilhas de estudo e exercícios.

## Objetivo

- Documentar minha evolução em Python.
- Organizar exercícios por contexto de aprendizado.
- Servir como trilha pública de estudo e prática contínua.

## Estrutura do Repositório

### faculdade

Atividades desenvolvidas no 1 periodo da faculdade, cobrindo práticas introdutórias e exercícios acadêmicos.

### leet_code

Atividades envolvendo desafios e práticas inspiradas em LeetCode, com foco em lógica e algoritmos.

### mundo_python_1

Conteúdos e exercícios do curso de Python 1 do professor Gustavo Guanabara.

### mundo_python_2

Conteúdos e exercícios do curso de Python 2 do professor Gustavo Guanabara.

Subpastas atuais:

- for
- if_else
- while

### udemy

Conteúdos e atividades do curso de Python do Luiz Otavio na Udemy (140 horas).

Subpastas atuais:

- args
- atividades

## Como Executar os Arquivos

1. Tenha o Python 3 instalado.
2. Abra o terminal na pasta do repositório.
3. Execute o arquivo desejado com:

```bash
python caminho/do/arquivo.py
```

Exemplo:

```bash
python mundo_python_1/aula_1.py
```

## Observações

- Alguns arquivos podem exigir bibliotecas externas.
- As atividades foram mantidas como trilha de estudo, podendo conter diferentes estilos conforme o momento de aprendizado.
